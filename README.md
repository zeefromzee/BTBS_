## Getting Started
Follow these steps to set up the project:

1. Clone the repository:
   ```bash
   git clone https://github.com/zeefromzee/BTBS_.git
   ```
2. Navigate into the project directory:
   ```bash
   cd BTBS_
   ```

## Requirements
- Python 3.x
- Required packages listed in `requirements.txt`
